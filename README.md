# Python-Algoritms
Collection of python algorithms and problem solving exercises.

# Features
- Modular python code
- simple command line tools
- well documented functions
- unit testing examples

# Installation

Clone the repository: 

Git clone https://github.com/christophersolaka/Python-Algoriyhm

Navigate into the project directory:

cd python-algoriym

install dependencies:

pip install -r requirements.txt

#usage 

run the calculator module:

python src/calculator.py

# project Structure

src/
Contains the main application source code.

tests/
Unit tests to verify program functionality.

docs/
Additional documentation and usage exaples.
